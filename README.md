## Portugues

### Sobre
Colecao de implementacoes de algoritmos classicos e desafios de programacao em JavaScript. Inclui algoritmos como Dijkstra, Fibonacci, Tribonacci, Collatz, verificacao de palindromos, ordenacao e mais.

### Tecnologias
- JavaScript

### Algoritmos incluidos
- Dijkstra (busca de caminho minimo)
- Fibonacci e Tribonacci
- Fatorial
- Collatz
- Palindromos
- Panagramas
- Numeros primos
- Conversao RGB-Hex
- Singleton pattern
- Flatten arrays
- E mais...

### Como rodar
Abra qualquer arquivo `.js` com Node.js:
```bash
node dijkstra.js
```

### Status
Repositorio de estudo, atualizado conforme novos algoritmos sao estudados.
